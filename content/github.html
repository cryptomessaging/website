Github Repositories

The following repositories provide a reference implementation of an iOS or Android messenger and the services to support it.

Just like web servers on the Internet, many different people and organizations can run message reflectors and make them available to any cryptomessaging client.

  • EdSig - Edwards-curve based HTTP authorization and certification SDK for Node.js and command line tools. Used by Cryptomessaging clients and services.
  • iOS and Android Messaging App - A React Native instant messenger using decentralized identity and reflectors.
  • Reflector Service - A reference implemention of a message reflector designed to run on AWS Lambda.
